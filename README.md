Как заполняем БД таблицами и прочим:
 - Создаем БД crm в psql  
 `CREATE DATABASE crm;`
 - Заходим в новую БД  
 `\connect crm`
 - Создаем под каждую подбазу в ней schema, для примера создадим схему под группу таблиц для аутентификации  
 `CREATE SHEMA auth;` 
 - Теперь нужно применить .sql файлы. Для той же auth из другой консоли будет так  
 `psql -U postgres -d crm -c "SET search_path TO auth;" -f AuthDB.sql`
 - Далее нужно сгенерировать код модели (если его еще нет) для Drogon через drogon_ctl :\)

   
А вообще, этим должен заниматься скрипт.
